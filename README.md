# Car Price Generator Report (Excel VBA Automation)

## Overview
This project is an Excel VBA automation system that generates car price reports from raw data.

The system creates pivot tables, charts, and exports reports automatically.

## Features
- Generate Pivot Table automatically
- Generate Chart automatically
- Export report to PDF
- Export data to CSV
- Print report
- Clear previous pivot and chart
- Validation before export (must generate pivot and chart first)
- Automatic refresh when new report is created
- Control button sheet for workflow management

## Dataset
- 250 rows dataset
- Car price data (2022)
- Average price analysis per manufacturer

## Workflow
Raw Data → Create Pivot → Create Chart → Validation → Export Report

## Tools
- Microsoft Excel
- VBA (Visual Basic for Applications)
